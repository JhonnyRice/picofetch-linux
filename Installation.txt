Image setup instructions
Download the snak.png file
place it in desired directory (mine is ~/Downloads/snak.png)


_____Program Install_______

1. Download the .sh file

2. Open The Terminal

3. Go to the directory the .sh file is (should be ~/Downloads)

4. (For the logo to display, if moved to different directory)

4,1: open the .sh file in your desired text editor (I use nano)

4,2: "tiv -h 25 -w 25 *~/Downloads/snak.png*
Change the path to where you have placed the image.
4,3: Save changes and exit to terminal

5. run: ./(name-here).sh
